# Arduino-Fire-Detector
Arduinoを使った火災報知器

レーザー加工機の排煙ケース内に設置して利用する為に作った物です。<br>
赤外線センサーを使っていて、蛍光灯や太陽光で誤動作するので、一般的な火災報知器としては使えない。<br>
